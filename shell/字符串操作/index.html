字符串操作

字符串修改

foo="hello"
echo $foo

把foo内容设置成hello。

foo+=" world"
echo $foo

为foo内容追加一个world，结果变成hello world

字符串替换

foo="hello world"
foo=${foo/o/0}
echo $foo

把第一个o换成0。输出：hell0 world

foo="hello world"
foo=${foo//o/0}
echo $foo

把所有o换成0。输出：hell0 w0rld

获取字符串长度

foo="hello"
echo ${#foo}

得到5。

截取字符串内容

# ${var:start:len}
far=123456
echo ${far:0:3}

从位置0开始截取3个字符，结果是123。

foo=/usr/bin/X11/startx
echo ${foo#*/}      # usr/bin/X11/startx
echo ${foo##*/}     # startx

从开始处截取内容。#表示截掉最短匹配字符串，##表示截掉最长匹配字符串。

bar=/usr/local/ect/local/networks
echo ${bar%local*}  # /usr/local/ect/
echo ${bar%%local*} # /usr/

从尾部开始截取内容。%表示截掉最短匹配字符串，%%表示截掉最长匹配字符串。
